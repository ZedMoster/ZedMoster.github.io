输入和输出的类型

Dynamo 节点可以将任何种类的数据作为输入，并可以输出任何种类的数据作为输出。

它能够将元素（例如 Revit 元素），抽象数字，字符串，Dynamo 几何图形以及其他所有内容作为输入。

但是，将 Revit 元素输入到 Python 节点时有一个警告。

Revit 元素需要进行转换类型后，才能通过 API 进行访问。
